Account Banking Debit External Collector

Keep a record of invoices sent to an external collector

This module generate a payment file for 'External Collector'.


The module creates two fields in the invoice: 'In collection' (boolean) and 'External Collector Ref.' (text).
These fields are completed when the payment file is generated. What allows to track and find the invoices sent to the external collector.


Notes
After installation, it is mandatory to create a payment mode called 'ext_collector' associated with the payment method 'External collector'.
