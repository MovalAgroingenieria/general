==========
Fax client
==========

Description
===========
Send and receive faxes by email using any service online.

After installation the drop list menu in Partners shows the option "Send Fax".


Configuration
=============
The only configuration required is the destination email (usually the chosen Fax service)
and as the sending email the email authorized by the service (it can be more than one).
Both are set in the mail template.
