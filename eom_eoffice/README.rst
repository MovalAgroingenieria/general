=================
Electronic Office
=================

Electronic Office with authentication based on digital certificate.

Description
===========

This module adds an electronic office, with DNIe-based authentication to enter
to Odoo frontend.

If the DNI of authenticated user is the same as the DNI of an existing partner,
then this partner and the authenticated user will be mapped.
