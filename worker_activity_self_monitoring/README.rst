===============================
Worker Activity Self Monitoring
===============================

**Table of contents**

.. contents::
   :local:


Description
===========

This module includes a monitoring solution for workers to know their activity
in real time. It is based on the Odoo's native activity log, but it is more
user-friendly and provides more information.
