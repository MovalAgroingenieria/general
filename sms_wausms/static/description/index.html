SMS WauSMS

Send SMS using WauSMS service

Send SMSs using Odoo's native modules, but refer to the cheaper WauSMS provider.


Configuration

Go to Settings > General Settings and select WauSMS as SMS service

Observe that the module SMS Alternatives has to be installed


Go to Settings > WauSMS and and fill in the necessary parameters.

You need to create an API password on the WauSMS website and authorise the public IP from where you send the SMS.
