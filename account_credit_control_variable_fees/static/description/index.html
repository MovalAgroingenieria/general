Account Credit Control Variable Fees

Add variable fees to credit control lines

This module add a new field 'variable fees' to credit control and other HTML field to explain the reason for these fees.





The application of these variable fees can be configured by Bank Journal. Thus, in the advanced configuration of a bank journal in the 'Miscellaneous' section you can find the 'Apply variable fees (Credit control)' check-box to apply or not the variable charges to the lines of this journal.
