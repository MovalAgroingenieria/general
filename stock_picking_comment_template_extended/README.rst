Description
===========

After installing this module, the different comment templates for stock
pickings are not added to the report by default. They have to be inserted by
clicking the **Insert comments** button. This action renders the templates for
the top and bottom comments and inserts them into two fields in the chosen order.
These fields can be edited without altering the templates.
