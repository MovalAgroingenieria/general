=================================
Account Adviser Config Permission
=================================

Description
===========

This module add new security group 'Adviser editor' for Accounting and Finance
category. Users with this groups are able to edit account config settings.

(*) Although the user with this security group can change any parameter, an
error message appears indicating that they do not have the appropriate
permissions. However, the changes are made. Ignore that warning message.
