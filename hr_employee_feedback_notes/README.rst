=======================
Employee feedback notes
=======================

Private Feedback Management for HR teams.

Description
===========

This module enables HR managers to record, organize, and manage private feedback notes for employees.
Each note includes its author, type (review, coaching, improvement plan, etc.), and is attached to a specific employee.
The notes are internal, structured, and accessible only to authorized personnel via configurable security groups.
It seamlessly integrates into the employee form, adding a dedicated tab for tracking development and follow-ups.
