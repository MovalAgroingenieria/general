Employee Attendances / Leaves Report

Summary of employee attendances, leaves and public holidays between a selectable date range

This module prints a summary report for attendances and leaves of employee or employees between selectable date range. The official holidays are also shown in other section.


Select the employees for whom you want to create the report, click on Action droplist and choose 'Employee Attendances / Leaves Report'.


After clicking on 'Employee Attendances / Leaves Report' a wizard opens that shows the selected employees, their number and allows you to select the date range for the report.
